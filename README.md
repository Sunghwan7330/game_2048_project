# game_2048_project

## 개요 

모바일 게임으로 유명한 2048 게임입니다. 

본 레퍼지토리의 목적은 go lang 과 TDD 를 연습하는 것입니다.

## 구성 

구성은 `board`, `screen`, `main` 으로 구분됩니다. 

## 테스트 

테스트는 아래의 명령어로 수행합니다. 

```shell
$ go test
PASS
ok      game_2048       0.014s
```

## 빌드 및 실행 

아래의 명령어로 빌드를 합니다. 

```shell
$ go build
```

빌드 후 생성된 `game_2048` 을 실행합니다.
